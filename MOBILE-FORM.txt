Mobile par Contact form Send karne ke liye
==========================================

Option 1 – Same WiFi (development)
----------------------------------
1. Computer par backend pehle chalao:  cd portfolio-backend  →  python app.py  (chalta rehna chahiye)
2. Naya terminal khol ke frontend chalao:  npm run dev  (portfolio ROOT folder se)
3. Terminal me "Network:" ke niche URL copy karo, jaise:  http://192.168.1.x:5173
4. Mobile ko SAME WiFi par connect karo, browser me bilkul yehi URL daalo (localhost ya 127.0.0.1 mat use karo)
5. Form bhar ke Send dabao

Agar ab bhi "failed to fetch" aaye:
- Confirm karo dono (backend + frontend) chal rahe hain aur mobile par Network wala URL hi open hai
- .env me VITE_API_URL mat set karo dev ke dauran (empty/comment out rakho)

Option 2 – Live deploy (kisi bhi device / mobile data se)
---------------------------------------------------------
1. Backend ko kahi host karo (e.g. Render, Railway, PythonAnywhere) – uski LIVE URL milegi
2. Root folder me .env file banao (ya .env.example copy karke .env banao)
3. .env me likho:  VITE_API_URL=https://your-actual-backend-url.com
4. Phir:  npm run build
5. Jo build nikle (dist/) use deploy karo (Vercel, Netlify, etc.)
6. Ab mobile ya koi bhi device se site kholo – form send hoga kyunki API ab LIVE URL use kar rahi hai

Check: Browser me open karke dekh lo – https://your-backend-url.com/api  – agar JSON dikhe to backend live hai.
